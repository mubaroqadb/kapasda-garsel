# KAPASDA  
**K**eusahaan **A**plikasi **P**enilaian **A**kapasitas **S**istem **D**aerah  
Sistem Penilaian Kesiapan Kecamatan Calon Daerah Otonom Baru (CDP) Kabupaten Garut Selatan

![KAPASDA](https://socialify.git.ci/mubaroqadb/kapasda-garsel/image?description=1&font=Inter&language=1&name=1&owner=1&pattern=Plus&theme=Light)

### Fitur Utama
- Input data per kecamatan (real-time hitung skor)
- Dashboard interaktif: ranking, chart, status layak/tidak layak
- Rekap lengkap + export Excel otomatis
- Admin: edit nilai pembanding, recalculate semua skor, backup data
- 100% offline-first (GitHub Pages)
- Tanpa desa (hanya fokus kecamatan)
- Tampilan modern, responsif, siap presentasi ke Bupati/DPRD

### Teknologi
- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript ES6 Modules (`.mjs`)
- Chart.js + SheetJS (XLSX)
- Supabase (auth + database)
